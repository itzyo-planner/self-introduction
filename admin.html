⏱️ 광고 시청 설정

AdSense 미연동 시 테스트 광고 시청 시간을 설정합니다.

기본값: 5초 (테스트/AdSense 미설정 시 적용)

📌 AdSense 연동 방법

1. adsense.google.com에서 승인 후 Publisher ID 발급
2. 광고 단위 → 리워드 광고(Rewarded Ads) 형식으로 새 광고 단위 생성
3. 슬롯 ID를 위 입력란에 입력하고 저장
4. index.html의 <script> adsbygoogle … 태그에 Publisher ID 추가 필요
5. AdSense 심사 통과 후 실제 광고가 표시됩니다. (미통과 시 테스트 모드 자동 적용)
